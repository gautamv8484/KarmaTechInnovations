# 🚀 KARMA Tech Innovations

![KARMA Tech Showcase](client/public/KARMA_Showcase.png) <!-- Just a placeholder if they add one -->

**KARMA Tech Innovations** is a premium, high-performance agency website built to showcase elite digital services including Web Development, App Development, and Graphic Design.

The platform is engineered with a cutting-edge "Modern Light Theme" utilizing Frosted Acrylic Glassmorphism, 3D WebGL interactions, and complex Parallax scroll animations to deliver a world-class user experience.

## ✨ Features
- **Immersive 3D Hero Environment:** Abstract floating geometric geometry built with `@react-three/fiber` and `Three.js`, complete with dynamic studio lighting and physical distortions.
- **Dynamic Project Portfolio Router:** Dedicated, data-driven `/projects/:id` portfolio internal pages showcasing deep technological stack breakdowns, galleries, and live links.
- **Scroll Spy Parallax & Interactions:** Advanced Framer Motion animations triggering on scroll—including asynchronous floating geometry, drawn vertical timelines, and metric count-ups.
- **Interactive Mouse Physics:** 3D card tilt tracking (`react-parallax-tilt`) creating elegant glare and depth simulating physical glass panels.
- **Performance Optimized:** Built on Vite with React 18 for blistering fast HMR and optimized production bundles.

## 🛠️ Tech Stack
- **Frontend Framework:** React 18, Vite
- **Styling:** Tailwind CSS (Custom Color System & Glassmorphism)
- **3D Rendering:** Three.js, React Three Fiber, React Three Drei
- **Animations:** Framer Motion, React-Parallax-Tilt
- **Routing:** React Router v6
- **Icons:** React-Icons

## 💻 Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/karma-tech-innovations.git
   ```
2. Navigate to the project directory:
   ```bash
   cd karma-tech-innovations/client
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173/` to view the website.
